Deploy AWS Spot Instances with a single command!

Quick Start

curl -sSL https://yourdomain.com/install.sh | bash -s -- create

Installation

No installation needed! Just run the commands directly:

1. Initial Setup

curl -sSL https://yourdomain.com/install.sh | bash -s -- setup

This will create a default configuration file at ~/.spot-deployer/config/config.yaml

2. Configure

Edit the configuration file with your settings:

nano ~/.spot-deployer/config/config.yaml

3. Deploy

curl -sSL https://yourdomain.com/install.sh | bash -s -- create

Commands

  • create - Create spot instances
  • destroy - Destroy all spot instances
  • list - List running instances
  • setup - Initial configuration setup
  • help - Show help message

Environment Variables

Set your AWS credentials:

export AWS_ACCESS_KEY_ID="your-key-id"
export AWS_SECRET_ACCESS_KEY="your-secret-key"
export AWS_DEFAULT_REGION="us-west-2"
Pro Tip: Use AWS SSO? The script automatically detects and uses your SSO credentials!

Advanced Usage

Use Specific Version

curl -sSL https://yourdomain.com/install.sh | bash -s -- create --version 1.0.0

Dry Run

curl -sSL https://yourdomain.com/install.sh | bash -s -- create --dry-run

Configuration

The configuration file supports:

  • Multiple AWS regions
  • Custom instance types
  • Tagging for organization
  • SSH key management
Security Note: Never commit your AWS credentials or private keys to version control!
